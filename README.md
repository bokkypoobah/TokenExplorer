# ERC-20, ERC-721 and ERC-1155 Token Explorer

https://bokkypoobah.github.io/TokenExplorer/

##### Requirements
* Web browser, with the MetaMask https://metamask.io/, or Rabby https://rabby.io/ browser plug-in
